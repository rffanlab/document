# utils帮助包使用方法，以及接口文档

## 使用方法(Maven)
```xml
		<dependency>
			<groupId>io.afu</groupId>
			<artifactId>utils</artifactId>
			<version>0.0.38-RELEASE</version>
		</dependency>
```
